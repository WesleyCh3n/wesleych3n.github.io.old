# Installation
## Install VcXsrv

```powershell
scoop install vcxsrv
```

## Setup xLauncher

```powershell
xlaunch.exe
```

## Add env variable to PowerShell PROFILE

```powershell
# vi $PROFILE, and add below line
$env:DISPLAY='localhost:0.0'
```
